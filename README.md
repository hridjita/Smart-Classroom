# Smart Classroom-A Classroom Management System
is a responsive classroom management web application designed to simplify student administration for educational institutions.

## Features

- Dashboard

- Student Registration

- Attendance Management

- Academic Results

- Announcements

- Responsive User Interface

- Local Storage Support

## Tech Stack

- HTML5

- CSS3

- JavaScript

- LocalStorage

1. Open the project in VS Code.

3. Install the Live Server extension.

4. Run `index.html` using Live Server.
